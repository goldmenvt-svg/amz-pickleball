Cấu hình GitHub Auto-Push

Khi kết nối, mỗi lần duyệt video sẽ tự động push lên GitHub → Vercel deploy → web cập nhật trong ~30 giây.

Cách tạo token (1 lần duy nhất):
1. Vào GitHub → Settings → Fine-grained tokens
2. Click "Generate new token"
3. Tên: AMZ Admin · Expiration: 90 days
4. Repository access: chọn amz-pickleball
5. Permissions → Contents: Read and write
6. Click "Generate token" → copy & paste vào ô trên
